# Sales Performance Analytics Dashboard

## Objective

Build an interactive Power BI dashboard to analyze sales performance
using Excel data and DAX.

## Tools

-   Power BI
-   Excel
-   DAX

## KPIs

-   Total Sales
-   Total Profit
-   Profit Percentage
-   Monthly Sales
-   Regional Performance

## DAX Measures

Total Sales = SUM(Sales\[Sales\])

Total Profit = SUM(Sales\[Profit\])

Profit % = DIVIDE(\[Total Profit\],\[Total Sales\])

## Dashboard Includes

-   KPI Cards
-   Line Chart (Monthly Sales)
-   Bar Chart (Region-wise Sales)
-   Product Table
-   Slicers
